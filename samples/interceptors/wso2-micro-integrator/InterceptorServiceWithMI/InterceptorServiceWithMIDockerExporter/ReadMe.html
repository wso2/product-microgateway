Docker Hello World Service

About

This sample leads you to build a docker image locally in your machine with a given target repository name and the target repository tag and push it to your docker repository.

Three modules get created with this sample. ESB Configs module contains all the ESB artifacts, the Composite Exporter module which is used to package and export the artifacts created within the ESB module, the Docker Exporter module which facilitates building and pushing of the docker image with the selected Composite Exporter modules (Capps).

ESB module contains a simple REST API called ‘HelloWorld’. When you invoke it, it generates a JSON message and sends it to the client.

  1. Configure docker image name

    - Open the POM.xml file in the Docker Exporter module and change the Docker image name by changing the Target Repository value and Target Tag value.

  2. Build the Docker Image

    - Open the pom.xml file and click the Build button in the top right corner.

    or

    - Right click the Docker Exporter project and select the Build Docker Image option.

  3. View the built docker image

    - Execute the following command in your terminal and you will see built docker image with given repository name and tag.

    								docker images

  4. Run the docker image

    - Execute the following command with the appropriate docker username, repository name and version.

    								docker run -p 8290:8290 -it docker_username/repository:v1.0.0

  5. Invoke the HelloWorld API

    Open the HTTP Client of WSO2 Integration Studio (If you don't see the HTTP Client pane, go to Window -> Show View - Other and select HTTP Client to enable the client pane).

    Enter the request information as given below and click the Send icon.

    Method:

    GET

    URL:

    									http://localhost:8290/HelloWorld

Open the pom.xml file and click the Push button in the top right corner.

or

Right click the Docker Exporter project and select the Push Docker Image option.

Both actions will open a wizard. Enter username and the password of your docker account and click on Push Image button.
